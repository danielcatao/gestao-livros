Sistema Integrado de Gestão de Livros

Projeto desenvolvido como requisito da disciplina de Programação Orientada a Objetos da Faculdade UniFECAF.

O desafio proposto é desenvolver um sistema de gerenciamento para uma biblioteca municipal que permita a administração eficaz de livro. O sistema deve possibilitar o cadastro de novos títulos, consultar livros, deletar um livro e atualizar um livro cadastrado. (CRUD)
